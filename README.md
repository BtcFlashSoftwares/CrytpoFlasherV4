# CryptoFlasherV4

**CryptoFlasherV4** is a blockchain transaction simulator designed to generate and send simulated Bitcoin (BTC) transactions across various wallet types and formats for educational and research purposes. It is a BTC fork designed to demonstrate how transaction data can propagate across the blockchain, allowing users to simulate high-volume BTC and USDT transactions.

### Features:
- **Blockchain Transaction Simulation**: Generate simulated BTC transactions viewable on the blockchain.
- **Wallet Compatibility**: Supports Segwit, Legacy, and other BTC address formats.
- **Cross-Wallet Transfers**: Simulated transactions are transferable across wallets.
- **Transaction Expiry**: Transactions remain visible for a set period (90 days for basic licenses, 360 days for premium licenses).
- **Transaction Limits**: Daily simulation limits of up to 100 BTC / 150,000 USDT (basic) and 500 BTC / 860,000 USDT (premium).

### Key Highlights:
- **Flash like real BTC is transferred**: These transactions are simulated for demonstration purposes.
- **Educational Use Only**: This tool is designed to help users understand how Bitcoin transactions propagate across the blockchain.
- **Blockchain Address Check**: Verify the destination address before transaction simulation.
- **Multiple License Options**: Basic and premium licenses available for extended features.
- **Full Confirmations**: Flash gets full confirmations like real crypto.
